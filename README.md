Каталог технологических стартапов

Веб-приложение для просмотра каталога технологических стартапов с возможностью добавления в избранное и создания заметок для инвестиционного анализа
Структура проекта

project/
├── index.html          # Основной HTML файл
├── style.css           # Стили приложения
├── script.js           # Логика приложения
└── README.md           # Документация
Структура данных

Каждый стартап содержит:

    ID (уникальный идентификатор)

    Название

    Описание

    Стадия развития

    URL веб-сайта
    Разработка

Для внесения изменений:

    Добавьте новые стартапы в массив startups в файле script.js

    Измените стили в файле style.css

    Обновите структуру в файле index.html
